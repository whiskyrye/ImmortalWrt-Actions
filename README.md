# ImmortalWrt编译
## 使用

- Fork 这个仓库
- 本地创建`.config`文件
- 上传`.config` 文件到 GitHub 仓库
- 在Actions页面选择`ImmortalWrt Builder`
- 点击 `Run workflow` 开始编译

## 添加turboacc

-带sfe:
 curl -sSL https://raw.githubusercontent.com/chenmozhijin/turboacc/luci/add_turboacc.sh -o add_turboacc.sh && bash add_turboacc.sh
